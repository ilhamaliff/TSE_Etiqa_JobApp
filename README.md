# Etiqa Trade Job Application System

An application system for job applications, integrated with Supabase Auth, PostgreSQL, and Storage for secure resume uploads.

## Tech Stack

- React
- Vite
- JavaScript
- Tailwind CSS
- Supabase (Auth, Database, Storage)


## How To Run

```powershell
npm install
npm run dev
```
